# AGENTS

Welcome to BigShot. This repo is organized so each module or script lives in its own folder with documentation for humans and bots.

## Getting Started
- Review the README in any folder to understand its intent.
- See `agent_notes.md` inside modules for running context.
- New code should be added in its own folder with README and AGENTS files.

### Folder Guide
- `modules/` — core features such as syncing and summarization
- `scripts/` — one-off utilities in subfolders
- `config/` — example configuration files
- `docs/` — architecture and developer docs
- `tests/` — placeholder test suite
- `ui/` — optional web UI components

Keep communication clear by updating agent notes when you modify functionality.
