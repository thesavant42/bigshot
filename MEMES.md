## Official Meme Policy Exception

This document establishes an **official exception** to BigShot's general no-emoji and no-meme policy for content specifically related to the OOPSIE scale and Windows path mishaps.

---

## 🚨 OOPSIE Exception Declaration 🚨

There is hereby a **global, universal exception** for the following content types when they relate to the OOPSIE scale and Windows path mishaps:

- **Memes** 🎭
- **GIFs** 🎬  
- **Emojis** 😅
- **Bardcore content** 🎵

This exception supersedes the standard policy outlined in `docs/DEVELOPER_GUIDE.md` and `docs/EMOJI_AUDIT.md` **exclusively** for OOPSIE-related discussions and documentation.

Contributors are **actively encouraged** to include humorous and dank meme content in the following contexts when relevant to OOPSIE:

- **GitHub Issues** discussing Windows path problems
- **Pull Requests** that fix path-related bugs
- **Code Comments** explaining path normalization logic
- **Documentation** related to cross-platform path handling
- **Discussion threads** about path separator mishaps

### Examples of Encouraged OOPSIE Content:

- "This backslash just caused an OOPSIE! 💥"
- GIFs of explosions when paths fail

- Emoji reactions to path-related bugs: 🤦‍♂️ 🔥 💀

## 🎯 Scope and Limitations

This exception is **strictly limited** to:
- Content directly related to file paths, directory separators, and cross-platform path handling
- Windows-specific path problems and their solutions
- Educational content about proper path normalization
- Humorous commentary on path-related debugging experiences

This exception does **NOT** apply to:
- General project discussions unrelated to paths
- Other types of bugs or issues
- Marketing or promotional content
- Off-topic memes or humor

---

## 🤝 Implementation Guidelines

When contributing OOPSIE-related meme content:

1. **Keep it relevant** - Content should genuinely relate to path handling issues
2. **Keep it constructive** - Humor should help explain or solve problems
3. **Keep it inclusive** - Avoid memes that might exclude or offend contributors
4. **Keep it fresh** - Prefer original content over overused memes
